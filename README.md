🚀 GitHub Issues Tracker
📌 Overview

A modern and responsive GitHub Issues Tracker web application that allows users to browse, search, and filter issues using real API data. The project focuses on clean UI design, smooth user experience, and practical JavaScript functionality.

✨ Features
Secure login system with predefined credentials
Fetch and display issues from API
Filter issues by All, Open, and Closed
Dynamic search functionality
Responsive card layout (4-column grid)
Clickable cards with modal for full issue details
Loading spinner for better user experience
Active tab highlighting
Conditional card border (Green for Open, Purple for Closed)
⚙️ API Endpoints

All Issues: https://phi-lab-server.vercel.app/api/v1/lab/issues

Single Issue: https://phi-lab-server.vercel.app/api/v1/lab/issue/{id}

Search Issue: https://phi-lab-server.vercel.app/api/v1/lab/issues/search?q=keyword

📚 JavaScript Concepts

Difference between var, let, and const

var is function scoped and can be re-declared and updated
let is block scoped and can be updated but not re-declared
const is block scoped and cannot be updated or re-declared

Spread Operator (...)
The spread operator is used to expand elements of an array or object into another array or object.

Difference between map(), filter(), and forEach()

map() returns a new array after transforming each element
filter() returns a new array based on a condition
forEach() executes a function for each element but does not return anything

Arrow Function
Arrow functions are a shorter syntax for writing functions in JavaScript.

Template Literals
Template literals allow embedding variables inside strings using backticks and ${}.
